# Password Generator (Tkinter GUI)

A simple password generator with a graphical interface built using Python and Tkinter.

## Features
- Choose how many letters, numbers, and symbols you want
- Generate and regenerate passwords
- Copy the generated password to the clipboard
- Input validation (minimum 1 letter, 1 number, 1 symbol)

## Requirements
- Python 3.x
- Tkinter (included with most Python installations)

## Run the app
```bash
python Generador_de_contrasena.py
